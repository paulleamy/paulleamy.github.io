Although, a large portion of these events can clearly be considered coughs (aspiration before expiration), a more substantial number of these events are more inline with thoat clearing sounds.

Using this webpage, audio files can be previewed from tabletop array recordings and headset recordings once. 1369 files are available for previewing and a specific file can be accessed using the "jump to" button below by entering a file number in the field. Associated annotations (which can be found here) will also show when a file is selected.
